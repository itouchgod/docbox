现在我们的文档保存系统已经完整设置好了：

1. 保存功能：
   - `savework` - 保存工作内容
   - `savestudy` - 保存学习内容
   - `savelife` - 保存生活内容
   - `savehobby` - 保存兴趣内容

2. 推送功能：
   - `pushall` - 一次性推送所有更改到 GitHub

使用流程：
1. 复制要保存的内容 (Cmd+C)
2. 根据内容类型使用对应命令，例如：
   ```bash
   savestudy    # 保存学习笔记
   ```
3. 需要推送时：
   ```bash
   pushall      # 推送所有更改
   ```

所有文件会按类别保存在对应目录，Git 提交会自动完成，推送时统一处理。
